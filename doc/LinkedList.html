LinkedList

Linked lists are used to implement other data structures because of its ability to efficiently add items to the start, middle, and end. A linked list is similar to a graph. You have nodes that point to other nodes. They look sorta like this: 1 -> 2 -> 3 -> 4 -> 5 Visualizing them as a JSON-like structure looks like this: { value: 1, next: { value: 2, next: { value: 3, next: {...} } } }
