Tree

DataStructure. Tree

Simple tree structure. It doesn't have any special rules to it and looks something like this:
Tree {
 root: {
  value: 1,
  children: [{
   value: 2,
   children: [...]
  },{
   value: 3,
   children: [...]
  }]
 }
}

Constructor

new Tree()

The tree has to start with a single parent, the "root" of the tree.
